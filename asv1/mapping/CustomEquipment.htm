AoEquipRestriction

This actor is used to prevent certain equipment being used on a mission. This can be found under Actor->Info->AoInfo->AoEquipRestriction. Simply place one down in your map and fill in the Restrict array with the classnames of the equipment you wish to restrict. This actor can also be used to allow only a certain quantity of each item. To do this, fill in the RestrictAmount array (this array corresponds to the Restrict classnames array you filled in before). There are also various restriction messages you can set here.

Note: It's probably a good idea to explain WHY the equipment is restricted in your mission briefing, so players aren't confused when they suddenly can't use an item they could use fine in previous missions.



AoCustomMapEquipment

This allows you to add custom equipment on a per map basis. It can be found under Actor->Info->AoInfo->AoCustomMapEquipment. Place one down in the map and set the properties:

CustomEquipment - this array holds custom primary/secondary equipment.
CustomExtraEquipment - this array holds custom extra equipment (third slot).

Note: These arrays should be populated with the AoGUIStarting class for the equipment, NOT the inventory class name.

For example, to make a custom weapon (say a rocket launcher), you would have to do the following:

  • Make your custom subclass of AoWeapon, e.g. AoWeaponRocketLauncher. Fill in its properties and code to behave how you wish (make sure to set the HUDIcon default property to a custom made graphic).
  • Then make a subclass of AoGUIStartingEquip called AoGUIStartingRocketLauncher. In its default properties, set InventoryClass to class'MyPackage.AoWeaponRocketLauncher' and set its FancyTitle and Description appropriately (these are the texts that will appear on the equipment selection screen).
  • Put down an AoCustomMapEquipment in your map and set CustomEquipment[0] to MyPackage.AoGUIStartingRocketLauncher.

Now when you start your map, you will see your custom equipment available in the list.
